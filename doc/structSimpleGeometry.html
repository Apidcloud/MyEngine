Detailed Description

template<typename T>
struct SimpleGeometry< T >

This is the simplest geometry. All faces are explicitly present and modifications are as simple as locating a face and modifying its values. A counter example would be the buffer geometry, which all faces have indexed values and thus requires extra work to modify a value.
