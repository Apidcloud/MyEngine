SoundObject& MyUPlay::MyEngine::SoundObject::play ( )

This function starts playing the audio, if this particular node is already active then it will restart the sound, nodes of the same file data will be unaffected.

If repeat is set to true when the sound ends, it will automatically start playing again.

SoundObject& MyUPlay::MyEngine::SoundObject::setAudio ( std::string  filename)

This function is expected to set this objects audio track, it also is expected to cache sounds its seen before and reuse their data instead of loading from disk every time (but only if the file name is exactly the same).

This function can throw a runtime_error if the file format is unsupported.
