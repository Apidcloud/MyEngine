template<typename T = float>
Vector3f MyUPlay::MyEngine::Renderer< T >::lastSortedPos
protected

This variable is for tracking the last known good sorting of objects for the camera. As long as the camera doesn't move beyond some epsilon and the scene doesn't have any new changes then we can skip sorting and use previous sortings. Rotation won't matter because it is distance sorted anyways. Frustum will use lastSortedRotation.
